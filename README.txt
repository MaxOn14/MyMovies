Проект приложения на системе андроид, получая данные с сайта MovieDB, они отображаются по рейтингу и просмотрам в приложении. Есть возможность переключаться
между вкладками топ рейтинга или топ оценок. Также можно добавлять фильмы в избранное. ВАЖНО: на данный момент сайт, с ккоторого берутся данные не работает с \
российскими IP, поэтому воспльзуйтесь vpn перед запуском приложения.
